## 📋 وصف المشروع

**أداة معلومات فيديوهات Telegram** هي تطبيق ويب خفيف الوزن تم تطويره لتحليل واستعلام البيانات الوصفية (Metadata) لمحتوى الفيديو **المتاح للعموم** في القنوات والمجموعات العامة على منصة Telegram. تهدف هذه الأداة إلى مساعدة المستخدمين والباحثين وأمناء الأرشيف الرقمي في الحصول على معلومات تقنية حول الفيديوهات—مثل العنوان، والتعليق التوضيحي، وتاريخ الرفع، وحجم الملف، والدقة المتاحة، ومدة التشغيل—دون التدخل في البنية التحتية للمنصة أو تجاوز أي آليات حماية.

### ✨ الميزات الرئيسية

- 🔍 **تحليل الروابط**: دعم إدخال عناوين URL لفيديوهات من قنوات/مجموعات Telegram العامة للاستعلام عن البيانات الوصفية المرتبطة بها
- 📊 **عرض المعلومات**: عرض واضح للعنوان، والتعليق، وتاريخ النشر، وحجم الملف، والدقة المتاحة، ومدة الفيديو
- 🌐 **واجهة عربية**: دعم كامل للغة العربية مع تصميم واجهة مستخدم احترافي وواضح للمستخدمين في المنطقة العربية والعالم
- 📱 **تصميم متجاوب**: تجربة مستخدم محسّنة لأجهزة الكمبيوتر المكتبية، والأجهزة اللوحية، والهواتف الذكية
- ⚡ **معالجة فعّالة**: التحقق من الصحة على جانب العميل مقترناً باتصال API مُحسّن لأوقات استجابة سريعة
- 🔒 **الخصوصية أولاً**: لا يتم تخزين أي بيانات للمستخدمين، أو سجلات الاستعلامات، أو محتوى الفيديو في أي مرحلة
