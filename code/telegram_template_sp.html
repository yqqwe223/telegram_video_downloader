✨ Funcionalidades principales y limitaciones

🎥 Descarga de videos de canales públicos

Admite descargar videos desde la versión web de canales/grupos públicos de Telegram (t.me/nombrecanal/IDmensaje), manteniendo la calidad original (hasta 1080P).

⚡ Sin necesidad de cuenta

No requiere número de teléfono de Telegram, clave API o credenciales de inicio de sesión, uso completamente anónimo, protege la seguridad de tu cuenta.

🛡️ Privacidad primero

Todas las análisis se completan en el lado del cliente, los enlaces directos de video no pasan por nuestros servidores, no registramos IP ni historial de descargas.

📱 Compatible con todas las plataformas

Adaptado para iOS, Android, Windows, macOS, funciona en navegadores principales (Chrome, Safari, Firefox).

✅ Compatible: Videos de versión web de canales/grupos públicos (enlaces t.me), contenido no protegido
❌ No compatible: Contenido de chats privados, grupos secretos, contenido protegido ("prohibido reenviar"), contenido exclusivo de Telegram Premium, conversión de GIF a video

⚠️ Limitación importante: Telegram no proporciona enlaces de video públicos estándar, esta herramienta no puede descargar directamente desde la aplicación de Telegram. Debe usar el enlace de versión web de canales/grupos públicos (formato como https://t.me/ChannelName/123).

📥 Tutorial detallado (tres sencillos pasos)

Paso 1: Obtener enlace de video de canal público de Telegram

Método A: A través de la aplicación de Telegram (recomendado)

  1. Abre la aplicación de Telegram, ingresa al canal o grupo público donde deseas descargar el video
  2. Mantén presionado el mensaje de video objetivo → toca ⋮ (más) en la esquina inferior derecha → selecciona "Copiar enlace"
  3. El formato del enlace debe ser:
    https://t.me/ChannelName/123
    o
    https://t.me/c/123456789/123

Método B: A través de Telegram Web

  1. Abre web.telegram.org en tu navegador
  2. Inicia sesión en tu cuenta de Telegram, encuentra el canal público
  3. Haz clic derecho en la marca de tiempo del mensaje de video → selecciona "Copiar dirección del enlace"

💡 Consejo importante: Asegúrate de que el canal/grupo esté en estado público (visible sin unirse). El contenido privado no se puede descargar.

Paso 2: Pegar enlace y analizar

  1. Regresa a esta página del descargador
  2. Pega el enlace t.me copiado en el cuadro de entrada superior
  3. Haz clic en el botón 【Analizar video】

El sistema automáticamente:
• Verificará si el enlace apunta a contenido público
• Extraerá metadatos del video (título, duración, miniatura)
• Generará enlace de descarga directa (formato MP4)
⏳ El proceso de análisis generalmente toma 3~8 segundos, dependiendo del tamaño del video

Paso 3: Descargar video HD

  • Una vez que el análisis tenga éxito, haz clic en el botón 【Descargar MP4】
  • El navegador comenzará automáticamente a descargar el archivo de video HD original
  • El archivo se guarda por defecto en la carpeta "Descargas" del dispositivo

📌 Sugerencias de uso:

  • ¿Descarga de archivos grandes interrumpida? → Usa un gestor de descargas (como IDM, Free Download Manager)
  • ¿Necesitas extraer audio? → Descarga MP4 y luego convierte a MP3 con herramientas en línea
  • ¿Guardar materiales importantes? → Se recomienda hacer también copia de seguridad de metadatos (título, enlace fuente)

⚠️ Nota: Los enlaces directos de video de Telegram generalmente tienen una validez de 1~2 horas. Completa la descarga oportunamente.

⚙️ Explicación del principio técnico

  • Esta herramienta analiza la estructura DOM de mensajes públicos de la versión web de Telegram y extrae enlaces directos de video (atributo src en la etiqueta <video>).
  • Para versiones más nuevas de Telegram, utiliza la configuración JSON de la página oficial t.me para extraer URL de medios, sin depender de ninguna API de terceros.
  • El video se proporciona en formato MP4 original, manteniendo la resolución completa y la tasa de bits al cargar, sin compresión, sin marca de agua.
  • Todo el procesamiento se completa en el lado del navegador: JavaScript obtiene directamente la URL del video y activa la descarga, sin pasar por ningún servidor intermedio.
  • Para contenido protegido (videos con configuración "prohibido reenviar"), Telegram cifra el flujo de medios, dicho contenido no se puede descargar con esta herramienta.

Explicación de limitaciones técnicas: Dado que Telegram no proporciona una API de descarga oficial y actualiza frecuentemente la estructura web, esta herramienta puede dejar de funcionar temporalmente después de una revisión de la versión web de Telegram. Prometemos actualizar oportunamente para mantener la compatibilidad.

❓ Preguntas frecuentes (FAQ)

P: ¿Por qué aparece "No se puede analizar el enlace"?
Posibles razones:
• El enlace no es de contenido de canal/grupo público (ej. enlace de chat privado)
• El canal tiene configurada protección de privacidad
• Actualización de estructura de la versión web de Telegram (lo repararemos lo antes posible)
• Formato de enlace incorrecto (debe comenzar con t.me/)
P: ¿Puedo descargar videos de grupos cifrados o chats privados?
No. Esta herramienta cumple estrictamente con las políticas de seguridad de Telegram, solo admite contenido completamente público, accesible sin inicio de sesión. El contenido privado requiere el cliente oficial de Telegram.
P: ¿Los videos descargados tienen marca de agua o reducción de calidad?
No. Proporcionamos el archivo de video original en los servidores de Telegram, idéntico en calidad a ver en la aplicación de Telegram, sin marcas de agua adicionales.
P: ¿Por qué no encuentro el archivo después de descargar en dispositivos iOS?
Limitación de iOS: Los archivos descargados en Safari se guardan en la aplicación "Archivos" → "iCloud Drive" → "Descargas". Se recomienda instalar la aplicación "Archivos" para gestionar el contenido descargado.
P: ¿Admite descargar documentos o archivos de audio de canales de Telegram?
La versión actual solo admite descarga de videos. ¡Estamos desarrollando soporte multi-formato (PDF, MP3, ZIP, etc.), espera nuestras futuras actualizaciones!
P: ¿Esta herramienta es segura? ¿Robará mi cuenta de Telegram?
Absolutamente segura. Esta herramienta no solicita ni recopila ninguna información de cuenta de Telegram (número de teléfono, sesión, contactos). Solo procesa enlaces de medios de páginas web públicas, completamente aislada de tu cuenta de Telegram.
