📥 Tutorial Penggunaan Detail (Unduh dengan 3 Langkah Mudah)

Langkah 1: Dapatkan Tautan Video Saluran Telegram Publik

Metode A: Melalui Aplikasi Telegram (Direkomendasikan)

  1. Buka Aplikasi Telegram, masuk ke saluran atau grup publik tempat Anda ingin mengunduh video
  2. Tahan lama pesan video target → Ketik titik tiga di kanan bawah ⋮ (Lainnya) → Pilih "Salin Tautan"
  3. Format tautan seharusnya:
    https://t.me/NamaSaluran/123
    atau
    https://t.me/c/123456789/123

Metode B: Melalui Telegram Web

  1. Buka web.telegram.org di browser
  2. Login ke akun Telegram Anda, cari saluran publik
  3. Klik kanan pada stempel waktu pesan video → Pilih "Salin Alamat Tautan"

💡 Tip Penting: Pastikan saluran/grup berstatus publik (dapat dilihat tanpa perlu bergabung). Konten pribadi tidak dapat diunduh.

Langkah 2: Tempel Tautan dan Parsing

  1. Kembali ke halaman pengunduh ini
  2. Tempel tautan t.me yang disalin ke kotak input di atas
  3. Klik tombol 【Parsing Video】

Sistem akan otomatis:
• Memverifikasi apakah tautan mengarah ke konten publik
• Mengekstrak metadata video (judul, durasi, thumbnail)
• Menghasilkan tautan unduhan langsung (format MP4)
⏳ Proses parsing biasanya membutuhkan 3~8 detik, tergantung ukuran video

Langkah 3: Unduh Video HD

  • Setelah parsing berhasil, klik tombol 【Unduh MP4】
  • Browser akan otomatis mulai mengunduh file video HD asli
  • File secara default disimpan ke folder "Unduhan" perangkat

📌 Saran Penggunaan:

  • Unduhan file besar terputus? → Gunakan pengelola unduhan (seperti IDM, Free Download Manager)
  • Perlu ekstrak audio? → Unduh MP4 lalu konversi ke MP3 menggunakan alat online
  • Menyimpan materi penting? → Disarankan untuk juga mencadangkan metadata (judul, tautan sumber)

⚠️ Perhatian: Tautan langsung video Telegram biasanya berlaku 1~2 jam, harap selesaikan unduhan tepat waktu.

⚙️ Penjelasan Prinsip Teknis

  • Alat ini menganalisis struktur DOM pesan publik versi web Telegram, mengekstrak tautan langsung video (atribut src dalam tag <video>).
  • Untuk Telegram versi yang lebih baru, gunakan konfigurasi JSON halaman t.me resmi untuk mengekstrak URL media, tidak bergantung pada API pihak ketiga apa pun.
  • Video disediakan dalam format MP4 asli, mempertahankan resolusi dan bitrate lengkap saat diunggah, tanpa kompresi, tanpa watermark.
  • Semua pemrosesan dilakukan di sisi browser: JavaScript langsung mendapatkan URL video dan memicu unduhan, tanpa melalui server perantara apa pun.
  • Untuk konten yang dilindungi (video dengan pengaturan "dilarang meneruskan"), Telegram mengenkripsi aliran media, konten semacam ini tidak dapat diunduh melalui alat ini.

Penjelasan Batasan Teknis: Karena Telegram tidak menyediakan API unduhan resmi, dan sering memperbarui struktur halaman web, alat ini mungkin sementara tidak berfungsi setelah pembaruan versi web Telegram. Kami berkomitmen untuk memperbarui secara tepat waktu untuk menjaga kompatibilitas.

⚠️ Kebijakan Privasi & Pernyataan Hukum

  1. Perlindungan Privasi: Alat ini tidak mengumpulkan, tidak menyimpan data pengguna apa pun. Semua parsing tautan dilakukan secara lokal di browser, tidak dikirim ke server.
  2. Hanya untuk Konten Publik: Dilarang keras mencoba mengunduh percakapan pribadi, saluran berbayar, atau konten yang tidak sah. Kami tidak menyediakan metode untuk melewati mekanisme perlindungan Telegram.
  3. Kepatuhan Hak Cipta: Konten yang diunduh hanya untuk penggunaan pribadi. Dilarang mengunggah ulang, distribusi komersial, atau melanggar hak pembuat. Hormati Ketentuan Layanan Telegram.
  4. Klausul Penafian: Pengguna harus menilai sendiri legalitas tindakan pengunduhan. Kami tidak bertanggung jawab atas pemblokiran akun, sengketa hukum yang disebabkan oleh penyalahgunaan alat ini.
  5. Tidak Ada Jaminan: Alat ini disediakan "sebagaimana adanya", tidak menjamin 100% kompatibel dengan semua video Telegram, tidak bertanggung jawab atas kehilangan data.

❓ Pertanyaan Umum (FAQ)

Q: Mengapa muncul pesan "tidak dapat memparsing tautan"?
Kemungkinan penyebab:
• Tautan bukan konten saluran/grup publik (misalnya tautan obrolan pribadi)
• Saluran mengatur perlindungan privasi
• Struktur versi web Telegram diperbarui (kami akan segera memperbaikinya)
• Format tautan salah (harus dimulai dengan t.me/)
Q: Dapatkah mengunduh video dalam grup terenkripsi atau obrolan pribadi?
Tidak bisa. Alat ini secara ketat mematuhi kebijakan keamanan Telegram, hanya mendukung konten yang sepenuhnya publik, dapat diakses tanpa login. Konten pribadi memerlukan akses melalui klien resmi Telegram.
Q: Apakah video yang diunduh memiliki watermark atau kualitas menurun?
Tidak ada. Kami menyediakan file video asli di server Telegram, kualitasnya sama persis dengan yang ditonton di Aplikasi Telegram, tanpa watermark tambahan apa pun.
Q: Mengapa di perangkat iOS setelah mengunduh tidak menemukan file?
Pembatasan iOS: File yang diunduh di Safari akan disimpan ke Aplikasi "File" → "iCloud Drive" → "Downloads". Disarankan untuk menginstal Aplikasi "File" untuk mengelola konten yang diunduh.
Q: Apakah mendukung pengunduhan dokumen atau file audio dalam saluran Telegram?
Versi saat ini hanya mendukung pengunduhan video. Kami sedang mengembangkan dukungan multi-format (PDF, MP3, ZIP, dll.), nantikan pembaruan selanjutnya!
Q: Apakah alat ini aman? Apakah akan mencuri akun Telegram saya?
Benar-benar aman. Alat ini tidak meminta, tidak mengumpulkan informasi akun Telegram apa pun (nomor telepon, sesi, kontak). Alat ini hanya memproses tautan media dari halaman web publik, sepenuhnya terisolasi dari akun Telegram Anda.
